# Capstone 3: Easy Shop API
A simple ecommerce website that allows users to create a profile and browse through products with a few search filters.

## Description
The application starts with a home screen where a wide range of products from electronics to clothes are displayed, a few price and category filters are available to use, and a login + profile page are customizable.

## Search Filters
### Price
Users are allowed to view products from a minimum and or maximum price range.

### Color
Users are also allowed to view products based on color

## User Profile
Users are allowed to log in their account and view their profile page. Here they can change any information including their name, address, and contact information.
